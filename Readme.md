# Traffic Classifier

## Installing Dependencies

* TensorFlow:
```
//with pip
pip install --upgrade TensorFlow
//with pip3
pip3 install --upgrade TensorFlow
```
* OpenCV:
```
//with pip
pip install opencv-python
//with pip3
pip3 install opencv-python
```
* Keras:
```
//with pip
pip install keras
//with pip3
pip3 install keras
```
